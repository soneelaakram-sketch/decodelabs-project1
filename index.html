About This Project

This project is centered around Semantic Integrity — mastering the core relationship between content and style before moving into dynamic, data-driven development.

The goal is to craft a clean, readable layout using only pure HTML markup and CSS styling logic — no frameworks, no shortcuts.

  • ✓ Semantic HTML5 tags used throughout
  • ✓ Single h1 per page rule followed
  • ✓ No inline styles — external CSS only
  • ✓ CSS Grid for layout, Flexbox for components
