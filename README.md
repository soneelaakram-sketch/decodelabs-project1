# DecodeLabs – Project 1: Static Webpage Design

**Batch:** 2026 | **Powered by:** DecodeLabs

## About
A static webpage built with semantic HTML and external CSS as part of the
DecodeLabs Industrial Training Kit. This project demonstrates foundational
frontend skills — clean structure, proper layout systems, and accessibility
standards.

## Tech Stack
- HTML5 (Semantic tags)
- CSS3 (Grid + Flexbox, CSS Variables)
- Google Fonts (Syne + DM Sans)

## Project Rules Followed
- One `<h1>` per page
- No inline styles — external CSS only
- DRY principle — no repeated declarations
- CSS Grid for page layout, Flexbox for components
- All images have explicit `width`, `height`, and `alt` attributes

## How to Run
1. Clone the repo
2. Open `index.html` in your browser — no build tools needed
